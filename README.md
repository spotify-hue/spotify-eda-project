# Spotify Top 200 EDA

This repository contains a Quarto project analyzing Spotify’s global Top 200 tracks (2017–2023).

## Files
- `EDA_Interpretation_Spotify.qmd` – interactive EDA in Python + Plotly
- `Spotify_Dataset_Cleaned-2(in) (1).csv` – dataset used
- `_quarto.yml` – Quarto site configuration
- `index.qmd` – project homepage

## How to render locally
```bash
quarto render
